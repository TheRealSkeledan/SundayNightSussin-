Sets an animated image over the screen. 
WARNING: Prone to crashes! 
Make sure the .xml has an animation named "idle"!!!

Value 1: Image 
ex. "Test" "fancy/BG1" "effects/epic/Screen Crack" 

Value 2: Scroll Factor (X, Y)
ex. "0.5, 0" "1.0, 0.5" 
(Note: 1.0 makes the background scroll at
the same speed as the characters.)

To delete the background, place another event
with an idential Value 1, and change value 2
to "delete" without quotes.
